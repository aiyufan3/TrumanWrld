# Product Rules

All automated optimizations must serve the TrumanWrld primary pipeline:
1. Signal Ingestion
2. Topic Ranking
3. Draft Generation
4. Risk Guard
5. Approval Queue
6. Publishing Abstraction
7. Analytics
8. Learning Loop

**Constraints**:
- Features must be V1 minimal viable engineering artifacts. Avoid over-engineered solutions.
- Human-in-the-loop is mandatory. Do NOT enable automated unsanctioned publishing mode.
