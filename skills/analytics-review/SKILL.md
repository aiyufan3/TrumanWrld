# Skill: Analytics Review

## Purpose
Record placeholders for execution metrics and ultimately tie into social impressions.

## When to Use
- Daily / Weekly chron jobs, or ad-hoc upon successful publish.

## Required Inputs
- Log events.

## Expected Outputs
- Generated internal markdown or console summary reports.

## Constraints
- Do not expose metric telemetry to external arbitrary APIs.

## Validation Checklist
- [ ] Can the analytics generate a readable daily output?
